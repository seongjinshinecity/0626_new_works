# 성수동 디저트 카페 경쟁 분석 (데일리브루 기준)

> 출처 가정: 데일리브루(자사) + 반경 500m 내 디저트 강점 카페 3곳을 공개 정보(메뉴/가격/리뷰 톤) 기준으로 정리한 데모 데이터.
> 자사 컨텍스트는 `../my-cafe-agent/my_cafe.md`(좌석 24석 / 인력 2명 / 주방 협소 / 디저트 강점)를 기준으로 한다.

## 1. 시장 개요
- 성수동은 카페 밀집 상권, 20~30대 여성·SNS 인증 수요가 핵심.
- 디저트(치즈케이크·크로플) 중심 객단가 ↑, 주말 웨이팅이 일상.
- 자사 VoC 분석 결과 **부정 리뷰 1위 = '대기시간'(주말 피크 집중)** → 회전율·동선이 경쟁 변수.

## 2. 경쟁사 비교표

| 항목 | 데일리브루(자사) | 어반로스터스 | 슈가힐 | 모닝릿 |
| --- | --- | --- | --- | --- |
| 콘셉트 | 디저트 강점 동네카페 | 스페셜티 로스팅 | 비주얼 디저트 전문 | 브런치+카페 |
| 시그니처 | 치즈케이크·크로플 | 싱글오리진 핸드드립 | 시즌 케이크 | 브런치 플레이트 |
| 객단가 | 9,000원 | 12,000원 | 11,000원 | 18,000원 |
| 좌석 | 24석(협소) | 40석 | 30석 | 55석 |
| 주말 웨이팅 | 길다(이슈) | 보통 | 매우 길다 | 보통 |
| SNS 강도 | 보통 | 약 | 강(인증샷) | 중 |
| 약점 | 좌석·대기 | 디저트 약함 | 가격·대기 | 커피 평범 |

## 3. 포지셔닝 요약
- **어반로스터스**: 커피 매니아층. 디저트 약함 → 자사 디저트가 우위.
- **슈가힐**: 비주얼·SNS 최강이나 가격↑·대기 심함 → 자사는 '맛+가성비'로 차별화 여지.
- **모닝릿**: 브런치 객단가↑·넓은 좌석 → 체류시간 경쟁, 디저트 카테고리는 비경쟁.

## 4. 자사 차별화 전략 (강점 살리고 약점 보완)
1. **'맛은 슈가힐급, 가격은 합리적'** 포지션 — 디저트 품질 호평을 메시지로 전면화.
2. **대기시간 약점 정면 대응** — 주말 모바일 예약/픽업 주문, 좌석 회전 동선 개선(VoC 1위 이슈 직접 해소).
3. **소규모(24석)의 강점 = 큐레이션** — 시즌 한정 디저트로 '여기서만' 경험 제공.

## 5. 추천 액션
- ⭐ **로컬 인플루언서 협업**(→ 프로젝트 9): 슈가힐 대비 약한 SNS 노출을 마이크로 인플루언서로 보완. '맛+가성비' 메시지를 인증샷으로 확산.
- 주말 예약/픽업 도입으로 대기 불만 완화 → 재방문율·평점 회복.
- 시즌 한정 디저트 캘린더 운영으로 재방문 동기 부여.
