# 타겟 인플루언서 정의 (데일리브루 기준)

> 출처: `../my-cafe-agent/my_cafe.md` (데일리브루 = 성수동 디저트 강점 동네 카페).
> 이 문서는 "누구에게 협업을 제안할지" 기준을 고정한다. 실제 후보 채우기는 `influencers.template.md`.

## 1. 우리 카페가 원하는 노출
- **핵심 강점**: 매장에서 직접 굽는 수제 **치즈케이크·크로플** (신선도·비주얼) → 사진/릴스로 잘 먹히는 디저트.
- **약점 보완 목표(8번 경쟁분석 연계)**: 슈가힐 등 비주얼·SNS 강한 경쟁사 대비 노출이 약함 → **마이크로/나노 인플루언서 다수**로 보완(7번 리서치 결론).
- **상권**: 성수동 + 20~30대 여성, 카공·주말 디저트 모임.

## 2. 타겟 인플루언서 조건

| 항목 | 기준 | 이유 |
| --- | --- | --- |
| 카테고리 | 카페/디저트/맛집/성수·서울 로컬 | 메뉴 적합성 |
| 팔로워 규모 | **1천~3만 (마이크로/나노 우선)** | 7번 리서치: 친근한 신뢰·높은 CTR·저비용 |
| 지역 | 서울, 특히 성수/성동 생활권 | 실제 방문 가능성 |
| 콘텐츠 톤 | 디저트 클로즈업·감성 사진/릴스 | 우리 비주얼과 합 |
| 참여(engagement) | **인게이지먼트율 ≥ 3%** (아래 공식) | 팔로워 수보다 실제 반응이 중요 |
| 광고 비중 | 협찬글이 과도하지 않을 것 | 신뢰도·진정성 |

## 3. 제외 기준
- 팔로워 대비 반응이 비정상적으로 낮음(가짜 팔로워 의심) → 인게이지먼트율로 거른다.
- 카테고리 불일치(여행/패션 일반 등 디저트와 무관).
- 협찬·광고만 도배.

## 4. 협업 형태 (예산 제약 반영)
- my_cafe.md 제약: **마케팅비 여력 적음** → 현금 협찬보다 **제품 시딩**(디저트 무상 제공 + 자발적 콘텐츠) 우선. (7번 리서치 'micro-influencer 제품 시딩' 결론과 일치)
- 1명 큰 협업보다 **여러 마이크로에 소액·시딩 분산**.
