# Top3 컨택 DM 초안 (템플릿)

> ⚠️ **사람이 직접 발송**합니다. 자동 DM·일괄 발송은 하지 않습니다(스팸·계정 정지 위험 + 안전 경계).
> `[ ]` 부분을 후보별로 채우세요. 진정성 있게 1명씩 개인화하는 것이 회신율을 높입니다.

## 공통 원칙
- 첫 메시지는 **짧게**, 칭찬 → 제안 → 부담 없는 마무리.
- my_cafe.md 강점(직접 굽는 수제 디저트)을 한 줄로.
- 예산 제약 → **제품 시딩(무상 디저트 + 자유로운 후기)** 중심 제안.

## 초안 A — 디저트 전문 인플루언서
```
안녕하세요 [@계정]님! 성수동에서 수제 디저트 카페 '데일리브루'를 운영하는 [이름]입니다 ☕
[최근 올리신 OO 게시물/릴스] 정말 잘 보고 있어요 — 디저트 담아내시는 감각이 저희 결과 잘 맞을 것 같아 연락드려요.
매장에서 직접 굽는 치즈케이크·크로플을 편하게 맛보시라고 초대하고 싶은데,
방문해 보시고 솔직한 후기 한 번 남겨주실 수 있을까 해서요. 부담 없이 편하게 봐주세요!
일정 맞춰볼 수 있으면 DM 주시면 감사하겠습니다 :)
```

## 초안 B — 성수/로컬 맛집 인플루언서
```
안녕하세요 [@계정]님, 성수동 골목 카페 '데일리브루'입니다!
[@계정]님 성수 맛집 소개가 동네 분들께 신뢰가 두텁더라고요. 저희도 동네에서 직접 굽는
수제 디저트로 알려가는 중이라, 한 번 들러 맛보시고 자연스럽게 소개해주시면 좋겠어요.
디저트는 저희가 준비해드릴게요. 관심 있으시면 편하게 답 주세요 :)
```

## 초안 C — 카공·감성 카페 인플루언서
```
안녕하세요 [@계정]님! 성수동 '데일리브루' 운영자 [이름]입니다.
콘센트·와이파이 갖춘 아늑한 자리에서 직접 구운 디저트와 커피 즐기실 수 있는 곳이에요.
[@계정]님 작업/감성 콘텐츠 분위기와 잘 어울릴 것 같아, 한 번 모셔서 디저트 대접하고 싶어요.
가볍게 둘러보시고 마음에 드시면 후기 부탁드릴게요. 좋은 하루 되세요!
```
