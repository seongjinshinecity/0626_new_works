# 인플루언서 후보 리스트 (템플릿 — 실데이터 입력 대기)

> ⚠️ **이 표는 빈 템플릿입니다.** 계정명·숫자는 **사용자가 본인 인스타 로그인 후 화면에서 확인한 실제 값**으로 채웁니다.
> Claude는 인플루언서 계정/팔로워/좋아요를 **지어내지 않습니다** — 검증되지 않은 데이터를 채우면 실제 사람을 잘못 표현하거나 가짜가 됩니다.
> 채우는 법: `methodology.md` 절차 → `engagement.py`로 율 계산 → 적합도 점수 → 정렬.
