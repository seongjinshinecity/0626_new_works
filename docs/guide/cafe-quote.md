# 가이드 — 카페 오픈 사입 견적 (쿠팡 리서치 → 노션 공유)

> 5주차 '카페 창업' — **오픈 준비(사입) 편**(🚩튜토리얼). 사입할 품목을 Playwright로 쿠팡에서 직접 검색해 가격을 모으고, 오픈 사입 견적서를 만들어 노션에 공유한다. 도구: Claude Code + Playwright MCP + Notion MCP.

## 왜 "진짜 브라우저"인가
- 쿠팡·네이버쇼핑은 검색 API가 없고 로그인·동적 로딩·로켓배송 가격이 섞여 있어 단순 web fetch로는 가격이 안 보인다.
- Playwright = 사람처럼 브라우저를 직접 띄워 검색하고 화면에서 가격을 읽는다.

## 설치
```
claude mcp add playwright npx '@playwright/mcp@latest'
# Notion MCP는 connectors에서 연결 (이미 연결돼 있으면 패스)
```
- 처음 실행 시 크롬 창이 뜸. 쿠팡이 로그인/캡차를 요구하면 그 창에서 직접 로그인한 뒤 다시 시킨다. (사람이 한 번 열어주고, 탐색은 AI가)

## 실습 A — 살 목록 정하기 (`shopping_list.md`)
하버 카페 오픈 사입 목록 예시: 원두 1kg 4봉 / 테이크아웃 종이컵 16oz 1000개 / 컵 리드 16oz 1000개 / 종이 빨대 1000개 / 냅킨 2000매 / 멸균 우유 1L 24팩 / 바닐라 시럽 1L 2병 / 음료 캐리어 2구 500개

## 실습 B — 쿠팡 가격 수집 → 견적서 (그대로 복붙)
> `shopping_list.md`의 품목을 Playwright로 쿠팡(coupang.com)에서 하나씩 검색해줘. 각 품목마다 로켓배송 대표 상품 1개의 이름·단가·상품 링크를 가져와줘(광고 상품은 거르고). 그 다음 `cafe_quote.md` 견적서를 만들어줘 — 표 컬럼은 품목 / 규격 / 수량 / 단가 / 금액(수량×단가) / 링크, 맨 아래 총 견적액 합계. 가격은 검색 시점 기준이라고 명시해줘.

- ✅ 기대 결과: 8개 품목이 채워진 견적표 + 총 견적액 합계.

## 실습 C — 노션에 공유 (그대로 복붙)
> 방금 만든 `cafe_quote.md` 견적서를 노션 페이지로 만들어 공유해줘. 제목은 "하버 카페 오픈 사입 견적"으로.

- ✅ 기대 결과: 노션에 견적 표가 그대로 들어간 페이지 생성. 동업자·투자자에게 링크만 보내면 끝.

## 팁 / 확장
- ① 쿠팡 vs 네이버쇼핑 가격 비교 컬럼 추가 ② 견적서를 document-skills로 엑셀(.xlsx)로 변환(수식·합계) → 운영 편과 연결 ③ `my_cafe.md`의 실제 사입 목록으로 바꾸면 만점.
- ⚠️ 사이트는 사람처럼 천천히. 과도한 자동 클릭·대량 스크래핑은 차단될 수 있음. 가격은 수집 시점 기준이라 실제 주문 전 한 번 더 확인.

> 유형: 🚩튜토리얼. Notion 원문에 별도 포인트 배점표는 표기돼 있지 않음.
