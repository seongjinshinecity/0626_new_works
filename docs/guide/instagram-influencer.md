# 가이드 — [홍보] 우리 카페 띄워줄 인스타 인플루언서 찾기 (20점)

> 우리 카페와 결이 맞는 인스타 인플루언서를 직접 찾아 협업 제안. 도구: Claude Code + Playwright MCP.

## 왜 사장님 일인가
- 딱 맞는 동네/감성 인플루언서 1명이 단골을 데려온다. 팔로워 수보다 **인게이지먼트율(좋아요+댓글÷팔로워)** + **타겟과의 결**이 핵심.

## 미션
Playwright로 **본인 인스타 로그인 세션**에서 후보 발굴 → 실제 게시물 데이터로 인게이지먼트율·적합도 계산 → Top 5 추천 + 컨택 메시지.

진행:
- 해시태그(#동네카페 #분위기좋은카페 #카페추천 #디저트맛집)·경쟁사 태그 게시물에서 후보 수집
- 후보별: 팔로워 / 최근 9개 평균 좋아요·댓글 / 인게이지먼트율 / 타겟 적합도(한 줄)
- `influencers.md` 에 Top5 표 + 상위3 DM 컨택 초안(무료 음료 체험 + 게시물 1건)

## 필수 제출물
1. 타겟 정의(지역/연령/감성, my_cafe.md 근거)
2. 인스타 탐색 실행 스크린샷
3. 후보 리스트 표(5명+): 계정/팔로워/인게이지먼트율/적합도
4. Top3 추천 + 컨택 초안
5. (가점) .xlsx/.pptx 정리 또는 노션 공유

> ⚠️ 안전: 본인 계정 로그인 세션, 사람처럼 천천히. 대량 자동 스크래핑·DM 자동 발송 금지(계정 정지 위험). 자동화는 발굴·열람까지, 컨택은 사람이 직접.
