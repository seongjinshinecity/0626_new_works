# 가이드 — 카페 월 손익 엑셀 + 1분기 PPT 보고서

> 5주차 '카페 창업' — **운영 보고 편**(🚩고블린). 한 달 장사 데이터로 월 손익을 엑셀로 정리하고, 동업자·투자자에게 보여줄 1분기 보고서를 PPT로 만든다. 준비: document-skills.
>
> 설치: `/plugin marketplace add anthropics/skills` → `/plugin install document-skills@anthropic-agent-skills`

## 실습 A — 한 달 장사 기록 → 월 손익 엑셀
- 입력: `cafe_q1.csv` (하버 카페 3~5월 매출·비용. 컬럼: 날짜,구분,카테고리,항목,금액). 샘플 데이터는 커리큘럼 원문에 포함.
- 요청 (그대로 복붙):
  > `cafe_q1.csv`로 월별 손익 엑셀 리포트를 만들어줘. 시트1=원본, 시트2=월별 요약(매출·비용·영업이익을 SUMIFS 수식으로), 카테고리별 매출 피벗, 월별 영업이익 막대 차트, 헤더 서식까지.
- ✅ 기대 결과: 매출 23,050,000 / 비용 15,630,000 / **영업이익 7,420,000**, 매월 우상향 차트.
- 💡 엑셀은 Excel·구글시트·Numbers로 열면 수식이 자동 계산됨. (맥 '미리보기/Quick Look'에선 수식 칸이 빈 칸으로 보일 수 있음 — 정상)

## 실습 B — 그 결과 → 1분기 보고 PPT
- 입력: `cafe_report.md` (배경 / 1분기 월별 실적표 / 카테고리별 매출 / 분석 / 추천 / 다음 액션). 샘플은 원문에 포함.
  - 월별: 3월 영업이익 2,010,000 / 4월 2,400,000 / 5월 3,010,000 / 합계 7,420,000
  - 카테고리: 음료 64% · 디저트 26% · 원두판매 10%
- 요청 (그대로 복붙):
  > `cafe_report.md` 내용을 5장 PPT 보고서로 만들어줘. 표지 / 배경·문제 / 분석(표+차트) / 추천 / 다음 액션. 깔끔한 톤으로.
- ✅ 기대 결과: 표지 + 4슬라이드, 분석 슬라이드에 월별 손익 표·차트, 추천에 '인플루언서 협업'(→ 홍보 퀘스트와 연결).

## 팁
- 두 실습은 같은 카페 데이터 — A에서 엑셀로 계산한 합계(7,420,000)가 B 보고서 표와 일치. **숫자 정리(엑셀) → 결론 보고(PPT)** 한 흐름.
- 데이터 없으면 샘플 그대로 따라 하고, `my_cafe.md`가 있으면 그 카페의 가상 한 달 매출로 바꿔본다.

> 유형: 🚩고블린. Notion 원문에 별도 포인트 배점표는 표기돼 있지 않음. (조사=리뷰·경쟁사 보고서, 홍보=인플루언서 찾기와 한 세트)
