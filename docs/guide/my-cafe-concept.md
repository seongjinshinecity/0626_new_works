# 가이드 — 내 카페 컨셉 정하기 (AI와 대화해서 my_cafe.md 만들기)

> 5주차 '카페 창업' 여정의 **출발점**(🚩퀘스트). AI와 대화하며 내 카페 컨셉을 정해 `my_cafe.md`로 박제한다. 이 파일이 이후 홍보·조사·운영 퀘스트 전부의 '내 데이터'(카페 버전 MISSION.md)가 된다.

## 배경 — 하버 카페 이야기
- 7년 차 마케터 지우 씨가 회사를 나와 골목 끝 10평 가게에 '하버 카페'("잠시 쉬어가는 항구")를 연다.
- 오픈 첫 주 하루 손님 11명, 옆 골목 스윗아워는 인스타 1.2만. 임대료·알바비·재료비는 매달 나간다.
- 막막한 일을 'AI 공장장'에게 하나씩 시킨다 → 5주차 카페 창업 퀘스트들이 이 도구들을 직접 만드는 과정.

## 실습 — 당신의 카페를 AI와 정하기
1. 클로드 코드에 그대로 복붙:
   > 너는 카페 창업 컨설턴트야. 내가 열고 싶은 카페 컨셉을 한 번에 하나씩 질문해서 구체화해줘. 순서는 ① 타겟 손님 → ② 위치·상권 → ③ 시그니처 메뉴 → ④ 분위기·인테리어 → ⑤ 가격대 → ⑥ 차별점 → ⑦ 카페 이름·슬로건. 내가 막히면 보기를 2~3개 제안해줘. 하나씩만 물어봐.
2. 질문에 솔직하게 답한다(정답 없음, 한 단어만 던져도 AI가 살을 붙임).
3. 완료 시 아래 형식으로 `my_cafe.md` 생성:
   - 카페 이름 / 한 줄 컨셉 / 타겟 손님 / 위치·상권 / 시그니처 메뉴 / 분위기·인테리어 / 가격대 / 차별점 / 슬로건

## 기대 결과 (예시 — 하버 카페)
```
# 내 카페: 하버 카페
## 한 줄 컨셉   작업과 수다 사이, 골목 끝 아늑한 동네 카페
## 타겟 손님    20~30대 여성. 혼자 노트북 작업 / 친구와 사진
## 위치·상권    주택가 골목, 조용. 근처 소형 오피스·대학
## 시그니처     제주 말차 말차라떼, 당일 구운 수제 치즈케이크
## 분위기       원목·따뜻한 조명·식물. 오래 머물고 싶은 느낌
## 가격대       아메리카노 4,500 / 시그니처 5,500
## 차별점       '머물러 작업하고 싶은 곳' + 사장님 친절
## 슬로건       "잠시 쉬어가는 항구, 하버 카페"
```

## 팁
- `my_cafe.md`는 카페 버전의 MISSION.md. 이후 튜토리얼·퀘스트(사입 견적 · 인플루언서 · 리뷰/경쟁사 · 손익 보고)에서 "하버 카페" 대신 이 파일을 근거로 시키면 전부 '내 카페' 맞춤으로 돌아간다.
- 여유 되면 메뉴판·로고 아이디어를 이미지로 뽑거나 `my_cafe.md`를 노션에 공유.

> 유형: 🚩퀘스트 (출발점). Notion 원문에 별도 포인트 배점표는 표기돼 있지 않음.
