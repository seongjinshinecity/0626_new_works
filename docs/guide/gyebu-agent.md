# 가이드 — [Agent+DB] 내 가계부 DB를 분석하는 에이전트

> 가계부 앱의 Supabase DB에 에이전트를 연결해 "말로 물어보는 소비 분석가"를 만든다.

## 전제
- gyebu-app 완료 + Supabase에 지출 10건 이상.

## 미션
- **Part 1 DB 연결**: 가계부와 같은 Supabase 프로젝트/테이블에 에이전트 접속(Supabase MCP)
- **Part 2 말로 물어보기**
  - 기본 조회: "이번 달 얼마 썼어?", "식비 최대일?", "교통비 월평균?"
  - 패턴 분석: "주중 vs 주말", "요일별 최대", "카테고리별 비율"
  - 절약 조언: "줄일 것 추천", "남은 예산", "연말 예상 지출"

## 핵심 구조
```
[가계부 앱으로 데이터 적재] → [에이전트가 Supabase 접속] → [질문] → DB 조회 + AI 분석 → [맞춤 답변]
```
- 앱: 적재 도구 · 에이전트: 활용 도구 · DB: 공유 자산

## 제출물
- GitHub 링크, 에이전트 대화 스크린샷(조회/분석/조언 각 1장+), 데이터 쌓인 앱 화면

## 팁
- 같은 Supabase 프로젝트/테이블에 연결. Supabase MCP로 직접 접속. 질문은 구체적으로.
