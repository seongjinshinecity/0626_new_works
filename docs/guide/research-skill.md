# 가이드 — [My Agent] 관심 분야 자동 리서치 스킬 만들기

> 웹 리서치 → md 정리 → 노션 공유 절차를 SKILL.md로 박제. 다음엔 "리서치 돌려줘" 한마디.

## 왜 스킬인가
- 한 번이면 프롬프트, 매주 반복+정해진 포맷이면 스킬. SKILL.md = "언제 + 어떻게"를 적어 내 방식대로 재현.

## 전제
- 브라우저 MCP(크롬/Playwright) 연결. Notion MCP 연결 시 보너스.

## 미션
- **Part 1 반복 주제 정하기** (채용/맛집/부동산/뉴스/중고/경쟁카페 등 자유)
- **Part 2 손으로 한 번 시켜보기**: 브라우저 MCP로 3곳+ 탐색 → research.md → 노션
- **Part 3 스킬 박제(핵심)**: `.claude/skills/research/SKILL.md` 저장. description=언제 쓰는지, 본문=절차(① 주제·키워드 질문 → ② 사이트 3곳+ 탐색 → ③ 포맷 md(제목/수집일/출처별 핵심/한 줄 요약/다음 액션) → ④ 노션 공유)
- **Part 4 재호출 검증**: 새 대화에서 "관심분야 리서치 해줘" → 같은 절차·포맷이면 성공(키워드 바꿔 2회)

## 핵심 구조
```
[한 번 손으로] 주제→탐색→md→노션  ──SKILL.md 박제──▶ 다음부턴 "리서치 돌려줘"로 매번 재현
```
> 에이전트는 한 번, 스킬은 매번.

## 제출물
- 주제·사이트 목록, 자동 탐색 스크린샷(3장+), SKILL.md, 재호출 결과 2개(동일 포맷), (보너스) 노션 업로드

## 포인트 획득 기준 (25pt)
| 항목 | 포인트 | 기준 |
|------|--------|------|
| 기본 완료 | 10pt | 웹사이트 3곳 이상 탐색 & md 파일 제출 |
| 스킬 박제 | 5pt | SKILL.md로 만들어 재호출이 실제로 동작 |
| 창의성 | 5pt | 자기만의 주제 선택 & md 정리 포맷이 실용적 |
| 공유 보너스 | 5pt | 단톡방에 결과물을 공유하고 다른 수강생 결과물에 리액션 |

- 달성 수준: 기본 제출(md만) **10pt** / 기본 + 스킬 박제·창의성 **15~20pt** / 올클리어(스킬 재호출까지 동작) **25pt**
