# 가이드 — [조사] 손님 리뷰·경쟁사 분석 → 엑셀/PPT 보고서 (20점)

> 리서치 결과를 회의에 바로 들고 갈 엑셀/PPT 보고서로 만든다. 준비: document 스킬.

## 미션
실무 에이전트 산출물 하나를 **.xlsx 또는 .pptx 보고서**로 완성.

### 경로 A — 리뷰(VoC) → 엑셀
`cafe_reviews.csv` (날짜,플랫폼,별점,리뷰,테마) 샘플로 VoC 분석 엑셀:
- 별점 분포 막대 차트 + 테마별 리뷰 수 피벗 + 부정 리뷰(별점 2 이하) 테마 Top3 + 헤더/서식
- 기대: 부정 1위 '대기시간' → "웨이팅 개선"을 가리킴

### 경로 B — 경쟁사 → PPT
`competitors.md` (시장 개요/경쟁사 비교표/분석/차별화/추천 액션) 샘플로 경쟁사 PPT:
- 표지 / 시장 개요 / 경쟁사 비교표 / 우리 차별화 / 추천 액션
- 기대: 추천에 '인플루언서 협업' 포함 → 홍보 편과 연결

> 샘플 데이터(cafe_reviews.csv 20행, competitors.md 비교표)는 커리큘럼 원문에 포함. 필요 시 그대로 복사해 사용.

## 필수 제출물
1. 원천 데이터(.md/.csv)
2. 변환된 .xlsx(수식·차트·서식) 또는 .pptx(표지·분석·추천·다음액션)
3. 생성 과정 + 결과 파일 열어본 실행 스크린샷
4. (가점) Notion/메일 공유

> my_cafe.md 기준으로 바꾸면 만점에 더 가까움. 두 경로 다 해보면 좋음.
