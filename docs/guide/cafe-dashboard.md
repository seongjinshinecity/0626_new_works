# 가이드 — [Auth+MCP+DB+App] 내 카페 사장님 대시보드 앱 (보스)

> Auth + MCP + DB + AI를 하나의 앱에 종합. 로그인하면 카페 데이터가 모이고 AI가 브리핑.

## 전제 (도전 퀘스트)
- 5주차 다른 퀘스트 먼저 완수 권장. my_cafe.md·운영 데이터 활용. **위젯 2~3개만 동작해도 인정.**

## 미션
- **Part 1 로그인/회원가입(Auth)**: 사장님만 접근
- **Part 2 데이터 소스 2개 이상 연결**: Notion MCP(할일/발주) / DB Supabase(매출·메뉴·재고) / 외부 API(날씨·뉴스)
- **Part 3 AI 브리핑**: 연결 데이터를 종합한 "오늘의 카페 브리핑" 생성
- **Part 4 대시보드 UI & 배포**: 한 화면 + Vercel

## 핵심 구조
```
[로그인] → [Auth 확인] → [Notion MCP + 카페 DB + 날씨 API] → [AI 종합] → [대시보드 표시]
```
- Auth: 접근 제어 · MCP: 외부 연동 · DB: 운영 데이터 · AI: 브리핑 생성

## 제출물
- 배포 URL(Vercel), GitHub 링크, 스크린샷(로그인→데이터 표시→AI 브리핑), 연결 소스 목록

## 팁
- 완벽 추구 금지, 매출 위젯 하나부터 점진 확장. 4주차 앱 코드 재활용. AI 브리핑은 "어제 매출+인기 메뉴 요약"만으로도 충분. 7주차 파이널 프로토타입으로 발전 가능.
