# 인스타 인플루언서 찾기 (프로젝트 9) — 스펙

## 목표
데일리브루(my_cafe.md)에 맞는 인스타 인플루언서를 **객관적 기준으로 발굴·선정·컨택**하는 체계를 만든다.

## ⚠️ 이 프로젝트의 성격: 프레임워크 + 반자동
실데이터 수집은 **사용자가 본인 인스타로 로그인**해야만 가능하다(계정 인증은 Claude가 대신할 수 없는 영역). 따라서:
- **Claude가 완성하는 것**: 타겟 정의, 선정 방법론(인게이지먼트율 공식 + 점수표), 빈 후보 템플릿, DM 초안, 보기 보조 스크립트, 계산기.
- **사용자가 하는 것**: 본인 로그인 → 해시태그 탐색 → 실제 후보/수치 입력 → Top5/Top3 확정 → 사람이 직접 DM.
- 상태 표기: **"프레임워크 완성 / 실데이터는 사용자 로그인 대기"** — ✅완료가 아님.

## 산출물
- `target.md` — 타겟 인플루언서 조건(my_cafe.md 기반)
- `methodology.md` — 인게이지먼트율 = (좋아요+댓글)/팔로워, 적합도 100점 점수표, 절차
- `influencers.template.md` — Top5 빈 표(실데이터 입력 대기, 가짜 금지)
- `dm-drafts.md` — Top3 DM 초안 3종(개인화 placeholder)
- `engagement.py` — 율/등급 계산기(검증됨)
- `assist_browse.py` — 본인 로그인 크롬에서 해시태그 한 명씩 열어주는 보기 보조

## 안전 경계 (반드시 준수)
- ❌ 인플루언서 계정·팔로워·좋아요 **지어내기 금지** — 가짜 후보 리스트는 최악의 산출물.
- ❌ 대량 자동 스크래핑 / 자동 DM / 자동 팔로우 금지.
- ❌ 비밀번호·세션 토큰 저장 금지(사용자가 직접 로그인).
- ✅ 보조 도구는 "한 명씩 보기"·"수치 입력 시 계산"만 — 수집/발송은 사람이.

## 검증
- `engagement.py`가 실제 수치 입력에 정확히 동작(6.96% 우수 / 0.25% 저반응 경고 확인).
- 템플릿이 my_cafe.md·7번 리서치(마이크로 시딩)·8번 경쟁분석(인플루언서 협업 추천)과 한 스토리로 연결.
