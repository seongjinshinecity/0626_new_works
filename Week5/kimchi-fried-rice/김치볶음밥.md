# 김치볶음밥 🍳

![김치볶음밥 썸네일](thumbnail/thumbnail.png)

> 잘 익은 김치 하나면 완성되는, 자취생부터 온 가족까지 사랑하는 국민 한 그릇.

**분량**: 1~2인분 · **조리 시간**: 약 15분 · **난이도**: ⭐️☆☆☆☆

---

## 재료

| 재료 | 분량 | 비고 |
|------|------|------|
| 밥 | 1.5공기 | 찬밥이 더 좋아요 |
| 잘 익은 김치 | 1컵 (약 150g) | 신김치일수록 감칠맛 ↑ |
| 김치 국물 | 2~3큰술 | |
| 대파 | 1/2대 | |
| 식용유 | 2큰술 | |
| 참기름 | 1작은술 | 마무리용 |
| 설탕 | 1/2작은술 | 신맛 중화 |
| 간장 | 1작은술 | 선택 |
| 계란 | 1~2개 | 반숙 프라이 |
| 김가루·통깨 | 약간 | 토핑 |

---

## 만드는 법

1. **재료 손질** — 김치는 한입 크기로 썰고, 대파는 송송 썰어요.
2. **파기름 내기** — 팬에 식용유를 두르고 대파를 넣어 약불에서 향을 내요.
3. **김치 볶기** — 썬 김치를 넣고 중불에서 3~4분 볶아요. 설탕을 넣으면 신맛이 부드러워져요.
4. **밥 넣기** — 밥과 김치 국물을 넣고 김치와 잘 섞이도록 볶아요. 간이 부족하면 간장을 살짝 추가해요.
5. **마무리** — 불을 끄고 참기름을 둘러 향을 더해요.
6. **계란 프라이** — 따로 반숙 계란 프라이를 부쳐 밥 위에 올려요.
7. **플레이팅** — 김가루와 통깨를 뿌리면 완성!

---

## 💡 꿀팁

- **찬밥 사용**: 갓 지은 밥보다 고슬고슬하게 볶여요.
- **스팸·참치 추가**: 단백질을 더하면 든든해요.
- **치즈 토핑**: 마지막에 모차렐라를 올려 녹이면 고소함 UP.
- **김치는 신김치로**: 익은 김치일수록 감칠맛이 깊어져요.
