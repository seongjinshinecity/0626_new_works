# 김치찌개 🍲

![김치찌개 썸네일](thumbnail/thumbnail.png)

> 푹 익은 신김치와 돼지고기로 끓이는, 밥 한 그릇 뚝딱하는 칼칼한 국물 요리.

**분량**: 2~3인분 · **조리 시간**: 약 30분 · **난이도**: ⭐️⭐️☆☆☆

---

## 재료

| 재료 | 분량 | 비고 |
|------|------|------|
| 잘 익은 김치 | 2컵 (약 300g) | 신김치일수록 맛 깊음 |
| 김치 국물 | 1/2컵 | |
| 돼지고기 (앞다리·삼겹) | 200g | 목살도 OK |
| 두부 | 1/2모 | |
| 대파 | 1대 | |
| 양파 | 1/2개 | |
| 청양고추 | 1개 | 칼칼함, 선택 |
| 다진 마늘 | 1큰술 | |
| 고춧가루 | 1큰술 | 색·맛 보강 |
| 국간장 | 1큰술 | |
| 설탕 | 1/2작은술 | 신맛 중화 |
| 참기름 | 1작은술 | 볶을 때 |
| 물 (또는 쌀뜨물) | 3컵 | 쌀뜨물이 더 진함 |

---

## 만드는 법

1. **재료 손질** — 김치는 한입 크기로, 돼지고기는 먹기 좋게 썰어요. 두부·양파·대파·고추도 썰어 둡니다.
2. **고기·김치 볶기** — 냄비에 참기름을 두르고 돼지고기를 볶다가, 김치와 고춧가루·다진 마늘을 넣고 3~5분 더 볶아 감칠맛을 끌어올려요.
3. **국물 붓기** — 물(또는 쌀뜨물)과 김치 국물을 붓고 센 불로 끓입니다.
4. **끓이기** — 끓어오르면 양파를 넣고 중불로 15분 정도 김치가 푹 무르도록 끓여요.
5. **간 맞추기** — 국간장·설탕으로 간을 맞춥니다. 신맛이 강하면 설탕을 조금 더해요.
6. **두부·마무리** — 두부, 대파, 청양고추를 넣고 5분 더 끓이면 완성!

---

## 💡 꿀팁

- **묵은지 사용**: 푹 익은 김치일수록 깊고 진한 맛이 나요.
- **쌀뜨물**: 물 대신 쓰면 국물이 한층 구수하고 부드러워져요.
- **돼지고기 먼저 볶기**: 기름과 감칠맛이 국물에 배어 훨씬 맛있어요.
- **참치·꽁치 버전**: 고기 대신 통조림을 넣으면 또 다른 별미.

맛있게 드세요! 😋
