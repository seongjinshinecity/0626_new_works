# 📅 QUARTER — 분기별 운영 캘린더

> 1년 = 4시즌. 분기마다 "새 카페 오픈"처럼 메뉴·공간·콜라보를 통째로 갈아엎는 운영 리듬.

---

## 한눈에 보는 4분기

| 분기 | 시즌 테마 | 시그니처 메뉴 방향 | 콜라보 디저트 | 공간·포토존 |
|---|---|---|---|---|
| **Q1 (1~3월)** | 겨울→봄, `RESET` | 따뜻한 시그니처 라떼 → 말차·딸기로 전환 | 봄 시즌 파티시에 콜라보 (딸기/벚꽃) | 화이트 메탈 + 봄빛 포인트 |
| **Q2 (4~6월)** | 초여름, `FRESH` | 시트러스·청량 에이드, 콜드브루 | 시그니처 케이크 (레몬/바질) | 통창·자연광 강조, 미니멀 그린 |
| **Q3 (7~9월)** | 한여름→초가을, `COOL` | 빙수·아이스 디저트 음료 | 여름 한정 젤라또/빙수 콜라보 | 스테인리스 질감 극대화, 차가운 무드 |
| **Q4 (10~12월)** | 가을→겨울, `WARM` | 밤·고구마·시나몬 따뜻한 음료 | 연말 홀케이크 콜라보 (예약 판매) | 블랙 + 골드 포인트, 무드 조명 |

---

## 분기 사이클 (3개월 표준 루틴)

각 분기는 아래 흐름을 반복한다.

### D-30 ~ D-15 (준비)
- [ ] 다음 시즌 테마·컬러 확정
- [ ] 신메뉴 2~3종 R&D, 시음 테스트
- [ ] 파티시에/브랜드 **콜라보 컨택 & 계약** (분기 1회)
- [ ] 포토존·플레이팅 시안 확정

### D-14 ~ D-3 (세팅)
- [ ] 신메뉴 레시피·원가·가격 픽스
- [ ] 직원 교육 (제조·플레이팅·세일즈 포인트)
- [ ] 공간 교체물(소품·포토존) 제작·설치
- [ ] SNS 티저 콘텐츠 예약 발행

### D-day (시즌 오픈)
- [ ] 신메뉴 + 콜라보 디저트 동시 출시
- [ ] 오픈 이벤트 (선착순/인증샷 혜택)
- [ ] 인플루언서·단골 초청 프리뷰

### D+1 ~ D+75 (운영)
- [ ] 주간 판매 데이터 체크 (베스트/사장 메뉴 정리)
- [ ] 콜라보 디저트 **한정 수량 소진 관리**
- [ ] 중간 SNS 콘텐츠 꾸준히 발행
- [ ] 다음 분기 테마 브레인스토밍 시작

### D+76 ~ D+90 (마감·인수인계)
- [ ] 시즌 결산 (매출·객단가·재방문율·인기 메뉴)
- [ ] 잘된 것/안된 것 회고 → 다음 분기 반영
- [ ] 다음 시즌 준비(D-30) 루틴으로 연결

## 연간 고정 이벤트

| 시기 | 이벤트 | 비고 |
|---|---|---|
| 분기 시작 | 시즌 오픈 데이 | 4회/년, 가장 큰 화제 타이밍 |
| 분기당 1회 | 파티시에 콜라보 디저트 | 한정 수량·희소성 |
| 연말 (Q4) | 홀케이크 예약 판매 | 객단가 최고 시즌, 사전예약 |
| 매 분기 말 | 시즌 결산 리포트 | 데이터 기반 의사결정 |

---

## 운영 팁

- **R&D 루틴화가 생명** — 분기마다 신메뉴+콜라보라 개발이 밀리면 전체 컨셉이 무너짐. D-30 준비를 절대 미루지 말 것.
- **콜라보는 분기 1회로 제한** — 희소성 유지 + 비용 통제. 매번 하면 특별함이 사라짐.
- **데이터로 다음 시즌 설계** — 결산에서 나온 베스트 메뉴 패턴을 다음 분기 메뉴 방향에 반영.
- **공간 교체는 포토존 중심** — 전체 인테리어를 바꾸는 게 아니라 소품·포토존만 갈아도 "새 카페" 느낌이 난다 (비용 효율).
