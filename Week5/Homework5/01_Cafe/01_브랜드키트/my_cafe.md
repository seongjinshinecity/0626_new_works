# ☕ QUARTER — 카페 컨셉 한 장 정리

**한 줄 정의** — 분기마다 메뉴·공간·디저트를 통째로 바꾸는, 블랙·스테인리스 메탈 미니멀 프리미엄 카페.

**슬로건** — *"시즌마다, 새로 짓다"*

---

| 항목 | 내용 |
|---|---|
| **① 타겟** | **메인** 대화·사진 손님(친구·연인·모임, 인스타) / **서브** 디저트 보고 오는 손님 |
| **② 위치** | 핫플 상권 (성수·연남·익선 등) — 노출 강함, 임대료·경쟁도 높음 |
| **③ 시그니처** | 시즌 한정 메뉴 — 계절마다 교체, 재방문·SNS 떡밥 |
| **④ 공간** | 미니멀 골격 + 블랙·스테인리스 소재 — 여백은 살리고 톤은 차갑고 시크하게 |
| **⑤ 가격** | 프리미엄 (아메리카노 6,500~8,000원) — 잔 퀄·플레이팅·서비스로 정당화 |
| **⑥ 차별점** | ⓐ "그 메탈 카페" 각인 + ⓑ 분기마다 메뉴·포토존까지 싹 교체 + ⓒ 분기 1회 파티시에 콜라보 한정 디저트 |
| **⑦ 이름** | **QUARTER** — 로고는 `Q` 한 글자 활용 |

---

## 운영 리듬 (핵심 엔진)

분기(3개월)마다 한 사이클이 돈다:

**새 시즌 메뉴 → 포토존·플레이팅 교체 → 파티시에 콜라보 디저트 출시 → SNS 화제 → 재방문**

→ 1년에 4번 "새 카페 오픈" 같은 이벤트가 터지는 구조.

## 강점

- 메탈 미니멀이라 인테리어 자체가 브랜드 → 사진 손님과 직결
- 분기 리뉴얼로 "질리는 핫플" 한계 극복, 단골 재방문 동력 확보
- 프리미엄 가격이 공간·콜라보·희소성으로 설명됨

## 주의할 리스크

- **메뉴 개발 부담** — 분기마다 신메뉴+콜라보, R&D 루틴화 필수
- **높은 기대치** — 프리미엄값 받는 만큼 잔 퀄·서비스 디테일 흔들리면 바로 평점 타격
- **임대료** — 핫플+프리미엄이라 초기 고정비 큼, 객단가·회전 설계 중요
