# QUARTER — 인트로 (2초 스플래시)

ASCII·글리치 모노크롬 인트로 애니메이션. 단일 `index.html`, 바닐라 JS + HTML5 Canvas (의존성 0).

## 동작
- 약 **2초** 동안 ASCII 노이즈 필드에서 **QUARTER** 워드마크가 글리치로 응결 → 정착
- 정착 후 슬로건 "시즌마다 새로짓다" + 가로 라인 페이드인
- 모노크롬(흑백), 스캔라인 + 비네팅 오버레이

## 인터랙션
- **마우스 이동** — 커서 주변 ASCII가 글리치로 흩어짐(리플)
- **클릭 / 스페이스 / 엔터** — 2초 인트로 리플레이
- 모바일: 터치로 글리치, 탭으로 리플레이
- 성능: 인트로 60fps → 정착 후 유휴 ~11fps로 스로틀(CPU 절약)

## 실행
```bash
cd 05_인트로
python3 -m http.server 8000   # → http://localhost:8000/
```
> Canvas 애니라 `file://`로 열어도 동작하지만, 폰트(Pretendard CDN) 때문에 로컬 서버 권장.

## 디자인 참고
핀터레스트 2장(ASCII 아트 / 흑백 브루탈리즘 타이포·픽셀 글리치) 기반 모노크롬 톤.
자세한 입력은 `command-input.txt`.
